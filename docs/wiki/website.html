Project Overview

The Coptic Aid Foundation website is being redesigned from scratch. The original site (copticaidfoundation.org) was built in 2013 and is outdated, inconsistent, and has no clear donor journey.

Goals

  • Primary: Explain the mission clearly
  • Secondary: Drive donations
  • Audience: Donors — many are older, accessibility is a priority

How to Update the Website

Working with Claude

  1. Start a Claude session and type /caf_skill — full project context loads
  2. Describe the change you want — Claude edits the HTML files locally
  3. Review in browser (open the file directly)
  4. When happy, say "push" — Claude commits and pushes to GitHub
  5. GitHub Pages deploys in ~1 minute. Hard refresh: Cmd+Shift+R

Global nav & footer

Edit docs/nav.html or docs/footer.html — then run the sync script or ask Claude to sync them to all pages.

Git workflow

  • Repo: github.com/saherwafai/CopticAidFoundation
  • Branch: main
  • GitHub Pages: served from /docs on main
  • Never rename or move the docs/ folder

Installing Claude skills (for new team members)

mkdir -p ~/.claude/skills/caf_skill
